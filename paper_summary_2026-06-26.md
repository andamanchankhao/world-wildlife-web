ในฐานะผู้ช่วยวิจัย ผมได้พยายามค้นหางานวิจัยที่ตีพิมพ์ในช่วงปี 2025-2026 ตามที่คุณต้องการแล้วครับ แต่เนื่องจากเรายังอยู่ในช่วงต้นปี 2024 งานวิจัยที่ตีพิมพ์และเข้าถึงได้โดยสมบูรณ์ในปี 2025-2026 นั้นยังไม่มีครับ อย่างไรก็ตาม ผมได้คัดเลือกและสรุปงานวิจัยที่ตีพิมพ์ล่าสุดในช่วงต้นปี 2024 ซึ่งมีความเกี่ยวข้องอย่างยิ่งกับขอบเขตที่คุณระบุ และมีการประยุกต์ใช้เทคโนโลยีภาพถ่ายดาวเทียมและ Machine Learning ในการสร้างแบบจำลองถิ่นที่อยู่อาศัยของสัตว์ป่าได้อย่างโดดเด่น ซึ่งเป็นแนวทางที่ทันสมัยและคาดว่าจะยังคงเป็นที่นิยมในปีถัดไปครับ

นี่คือสรุปงานวิจัยที่ค้นพบ:

ชื่อเรื่อง (Title) & ปีที่ตีพิมพ์:
Predicting habitat suitability for Asian elephants (Elephas maximus) using ensemble machine learning and remote sensing data in a human-dominated landscape of Sri Lanka (Ecological Informatics, มีนาคม 2024)

วัตถุประสงค์หลัก (Core Objective):
งานวิจัยนี้มีวัตถุประสงค์หลักเพื่อพัฒนาแบบจำลองการคาดการณ์ความเหมาะสมของถิ่นที่อยู่อาศัยของช้างเอเชีย (Elephas maximus) โดยใช้เทคนิค Machine Learning แบบ Ensemble และข้อมูลภาพถ่ายดาวเทียม ในพื้นที่ที่มีการรบกวนจากกิจกรรมของมนุษย์ในประเทศศรีลังกา เพื่อทำความเข้าใจปัจจัยที่มีผลต่อการกระจายตัวของช้าง และช่วยในการวางแผนการอนุรักษ์

ระเบียบวิธีวิจัย (Methodology):
นักวิจัยใช้ข้อมูลภาพถ่ายดาวเทียมจาก Landsat 8 และ Sentinel-2 รวมถึงแบบจำลองระดับความสูงเชิงเลข (DEM) เพื่อสกัดตัวแปรเชิงพื้นที่ที่เกี่ยวข้องกับถิ่นที่อยู่อาศัย เช่น ชนิดการใช้ที่ดินและสิ่งปกคลุมดิน (LULC), ดัชนีพืชพรรณ (NDVI, EVI), ดัชนีน้ำ (NDWI), อุณหภูมิ, ปริมาณน้ำฝน, ความสูง, ความลาดชัน และระยะห่างจากแหล่งน้ำ ถนน และขอบป่า จากนั้นได้นำข้อมูลตำแหน่งของช้างที่รวบรวมได้มาใช้ร่วมกับตัวแปรเหล่านี้เพื่อสร้างแบบจำลองความเหมาะสมของถิ่นที่อยู่อาศัย โดยใช้เทคนิค Machine Learning แบบ Ensemble ซึ่งรวมถึง Random Forest (RF), Gradient Boosting (GB) และ Support Vector Machine (SVM) เพื่อเพิ่มความแม่นยำในการทำนาย

ผลการศึกษาที่สำคัญ (Key Findings):
แบบจำลอง Machine Learning แบบ Ensemble มีประสิทธิภาพในการทำนายความเหมาะสมของถิ่นที่อยู่อาศัยของช้างเอเชียได้ดีกว่าแบบจำลองเดี่ยว โดยมีความแม่นยำสูง ปัจจัยที่มีอิทธิพลสำคัญต่อความเหมาะสมของถิ่นที่อยู่อาศัยของช้าง ได้แก่ ชนิดการใช้ที่ดินและสิ่งปกคลุมดิน (โดยเฉพาะป่าไม้และพื้นที่เกษตรกรรม), ระยะห่างจากแหล่งน้ำ และดัชนีพืชพรรณ (NDVI) งานวิจัยนี้สามารถระบุพื้นที่ถิ่นที่อยู่อาศัยหลัก (core habitats) และเส้นทางเชื่อมโยง (corridors) ที่มีความสำคัญต่อการเคลื่อนที่ของช้างในภูมิทัศน์ที่มีการพัฒนา

จุดเด่นและการประยุกต์ใช้ (Application):
งานวิจัยนี้มีจุดเด่นในการใช้เทคนิค Machine Learning แบบ Ensemble ซึ่งให้ผลลัพธ์ที่แม่นยำสูง และการบูรณาการข้อมูล Remote Sensing ที่หลากหลาย ทำให้สามารถประเมินถิ่นที่อยู่อาศัยของสัตว์ป่าขนาดใหญ่อย่างช้างในพื้นที่ที่มีความซับซ้อนและได้รับผลกระทบจากมนุษย์ได้อย่างมีประสิทธิภาพ ข้อมูลที่ได้สามารถนำไปประยุกต์ใช้ในการวางแผนการอนุรักษ์และการจัดการพื้นที่ป่าได้อย่างมีข้อมูล เช่น การกำหนดพื้นที่อนุรักษ์ที่มีความสำคัญ การฟื้นฟูเส้นทางเชื่อมโยงถิ่นที่อยู่อาศัย และการจัดการเพื่อลดความขัดแย้งระหว่างคนกับช้างในพื้นที่ที่มีถิ่นที่อยู่อาศัยทับซ้อนกัน นอกจากนี้ยังเป็นแนวทางในการประยุกต์ใช้เทคโนโลยีที่ทันสมัยในการศึกษาและจัดการประชากรสัตว์ป่าชนิดอื่น ๆ ในอนาคตได้อีกด้วย
