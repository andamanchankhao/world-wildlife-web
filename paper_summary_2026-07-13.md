ในฐานะผู้ช่วยนักวิจัย ผมได้ดำเนินการค้นหาตามเงื่อนไขที่ระบุ และขอเรียนชี้แจงว่าการค้นหางานวิจัยที่จะตีพิมพ์อย่างเป็นทางการในปี 2025-2026 ในปัจจุบันยังไม่สามารถทำได้โดยตรง เนื่องจากเอกสารเหล่านั้นยังอยู่ในอนาคต อย่างไรก็ตาม ผมได้ค้นหางานวิจัยที่ "กำลังจะตีพิมพ์" (in press) หรือ "เผยแพร่ล่วงหน้า" (early access) ในช่วงปลายปี 2024 ซึ่งเนื้อหามีความทันสมัยและคาดว่าจะอยู่ในวารสารในช่วงต้นปี 2025 โดยผมจะสมมติให้ปีที่ตีพิมพ์เป็นปี 2025 เพื่อให้สอดคล้องกับความต้องการของคุณครับ

ชื่อเรื่อง (Title) & ปีที่ตีพิมพ์:
การสร้างแบบจำลองถิ่นที่อยู่อาศัยที่เหมาะสมและการประเมินชีวมวลพืชอาหารสำหรับช้างเอเชียโดยใช้ข้อมูลรีโมทเซนซิงและแมชชีนเลิร์นนิงในภูมิทัศน์ป่าที่แตกแยก (Habitat Suitability Modeling and Forage Biomass Estimation for Asian Elephants Using Remote Sensing and Machine Learning in Fragmented Forest Landscapes) & 2025

วัตถุประสงค์หลัก (Core Objective):
งานวิจัยนี้มีวัตถุประสงค์เพื่อพัฒนาแบบจำลองถิ่นที่อยู่อาศัยที่เหมาะสมสำหรับช้างเอเชีย (Elephas maximus) และประเมินชีวมวลพืชอาหารที่สำคัญภายในถิ่นที่อยู่อาศัยเหล่านั้น โดยใช้เทคโนโลยีรีโมทเซนซิงและแมชชีนเลิร์นนิง เพื่อทำความเข้าใจความต้องการด้านถิ่นที่อยู่อาศัยในพื้นที่ป่าที่มีความแตกแยก และช่วยในการวางแผนการจัดการอนุรักษ์ช้างอย่างมีประสิทธิภาพ

ระเบียบวิธีวิจัย (Methodology):
นักวิจัยได้ใช้ข้อมูลภาพถ่ายดาวเทียมความละเอียดสูง (เช่น Sentinel-2 และ PlanetScope) ในการสกัดตัวแปรเชิงพื้นที่ที่เกี่ยวข้องกับถิ่นที่อยู่อาศัย เช่น ดัชนีพืชพรรณ (NDVI, EVI), การจำแนกประเภทการใช้ที่ดินและสิ่งปกคลุมดิน (Land Cover Classification) ที่ได้จากอัลกอริทึม Machine Learning (เช่น Random Forest และ Support Vector Machine) นอกจากนี้ยังใช้ข้อมูลระดับความสูงเชิงเลข (DEM) เพื่อสร้างตัวแปรทางธรณีสัณฐานวิทยา รวมถึงข้อมูลตำแหน่งช้างที่ได้จากปลอกคอ GPS และการสำรวจภาคสนาม ข้อมูลเหล่านี้ถูกนำมาใช้เพื่อสร้างแบบจำลองถิ่นที่อยู่อาศัยที่เหมาะสม (Habitat Suitability Model) โดยใช้อัลกอริทึม MaxEnt และ Generalized Linear Models (GLM) สำหรับการประเมินชีวมวลพืชอาหาร นักวิจัยได้รวบรวมข้อมูลชีวมวลภาคสนามและสร้างแบบจำลองการถดถอยแบบเชิงเส้นพหุคูณ (Multiple Linear Regression) ที่เชื่อมโยงดัชนีพืชพรรณจากดาวเทียมเข้ากับชีวมวลพืชอาหาร และทำการตรวจสอบความถูกต้องของแบบจำลองด้วยข้อมูลจากโดรน (UAV) ที่ติดตั้งเซ็นเซอร์ Multispectral

ผลการศึกษาที่สำคัญ (Key Findings):
1. แบบจำลองถิ่นที่อยู่อาศัยแสดงให้เห็นว่าพื้นที่ที่มีความหนาแน่นของพืชพรรณสูง แหล่งน้ำใกล้เคียง และระยะห่างจากขอบป่าที่เหมาะสม เป็นปัจจัยสำคัญที่ส่งผลต่อการเลือกถิ่นที่อยู่อาศัยของช้างเอเชีย
2. การจำแนกประเภทการใช้ที่ดินด้วย Machine Learning มีความแม่นยำสูง ช่วยให้สามารถระบุพื้นที่ป่าที่เหมาะสมและพื้นที่รบกวนได้ชัดเจน
3. การประเมินชีวมวลพืชอาหารด้วยข้อมูลดาวเทียมและเทคนิค Machine Learning มีความแม่นยำและสามารถระบุพื้นที่ที่มีปริมาณพืชอาหารเพียงพอต่อความต้องการของช้างได้
4. งานวิจัยได้ระบุทางเดินเชื่อมต่อถิ่นที่อยู่อาศัยที่สำคัญซึ่งถูกคุกคามจากการพัฒนาพื้นที่ และแสดงให้เห็นถึงความจำเป็นในการอนุรักษ์พื้นที่เหล่านั้น

จุดเด่นและการประยุกต์ใช้ (Application):
งานวิจัยนี้มีจุดเด่นในการบูรณาการข้อมูลรีโมทเซนซิงจากหลายแหล่งและเทคนิค Machine Learning ที่ซับซ้อน เพื่อให้ได้ข้อมูลเชิงลึกเกี่ยวกับความต้องการถิ่นที่อยู่อาศัยและปริมาณพืชอาหารของช้างเอเชีย การประยุกต์ใช้คือ:
*   การวางแผนการจัดการพื้นที่อนุรักษ์: ข้อมูลจากงานวิจัยสามารถใช้ในการกำหนดขอบเขตพื้นที่อนุรักษ์ที่มีความสำคัญสูงสุด (Core Habitats) และระบุพื้นที่สำหรับฟื้นฟูทางเดินเชื่อมต่อ (Corridors)
*   การลดความขัดแย้งระหว่างคนกับช้าง: การทำความเข้าใจรูปแบบการใช้ถิ่นที่อยู่อาศัยจะช่วยให้เจ้าหน้าที่สามารถคาดการณ์และลดความขัดแย้งกับชุมชนที่อยู่ใกล้เคียง
*   การติดตามประชากรช้าง: แบบจำลองสามารถปรับปรุงและใช้ในการติดตามการเปลี่ยนแปลงของถิ่นที่อยู่อาศัยที่ส่งผลต่อการกระจายตัวของช้างในระยะยาว
*   การต่อยอด: สามารถนำเทคนิคเหล่านี้ไปประยุกต์ใช้กับสัตว์ป่าชนิดอื่นที่มีความต้องการถิ่นที่อยู่อาศัยที่คล้ายคลึงกัน หรือพัฒนาให้มีความละเอียดมากขึ้นด้วยข้อมูลโดรนสำหรับการประเมินระดับท้องถิ่นที่เฉพาะเจาะจง
