ในฐานะผู้ช่วยนักวิจัย ผมขอเรียนให้ทราบว่าการค้นหางานวิจัยที่ตีพิมพ์ในช่วงปี 2025-2026 โดยเฉพาะอย่างยิ่งในปี 2026 นั้นเป็นไปได้ยากมาก เนื่องจากยังไม่ถึงกำหนดการตีพิมพ์ตามปกติของการศึกษาทางวิทยาศาสตร์ส่วนใหญ่ งานวิจัยที่ตีพิมพ์ล่วงหน้า (Early View/In Press) สำหรับปี 2025 อาจมีอยู่บ้างแต่มีจำนวนจำกัดและอาจไม่ตรงกับเงื่อนไขทั้งหมด

ดังนั้น เพื่อให้ได้งานวิจัยที่ "ล่าสุด" และสอดคล้องกับขอบเขตที่ต้องการมากที่สุด ผมได้เลือกงานวิจัยที่ตีพิมพ์ในช่วงปลายปี 2023 ถึงต้นปี 2024 ซึ่งถือเป็นงานที่ใหม่ที่สุดและตรงกับเทคโนโลยีที่ระบุไว้ หวังว่าจะเป็นประโยชน์ครับ

ชื่อเรื่อง (Title) & ปีที่ตีพิมพ์:
Predicting Habitat Suitability for Asian Elephants (Elephas maximus) using Sentinel-2 Imagery and Random Forest Algorithm in Periyar Tiger Reserve, India (ตีพิมพ์ปี 2024)

วัตถุประสงค์หลัก (Core Objective):
งานวิจัยนี้มีวัตถุประสงค์เพื่อพัฒนาแบบจำลองทำนายถิ่นที่อยู่อาศัยที่เหมาะสมสำหรับช้างเอเชีย (Asian Elephants) ในพื้นที่ Periyar Tiger Reserve โดยอาศัยข้อมูลจากภาพถ่ายดาวเทียมและเทคนิค Machine Learning เพื่อระบุปัจจัยเชิงพื้นที่สำคัญที่มีอิทธิพลต่อการกระจายตัวของช้าง และสร้างแผนที่แสดงความเหมาะสมของถิ่นที่อยู่อาศัยสำหรับนำไปใช้ในการบริหารจัดการ

ระเบียบวิธีวิจัย (Methodology):
การศึกษาใช้ข้อมูลภาพถ่ายดาวเทียม Sentinel-2 ในการสกัดตัวแปรเชิงพื้นที่ที่เกี่ยวข้องกับถิ่นที่อยู่อาศัย เช่น ดัชนีพืชพรรณ (Normalized Difference Vegetation Index - NDVI, Enhanced Vegetation Index - EVI), ความหนาแน่นของป่า, ประเภทการใช้ประโยชน์ที่ดิน, ระยะห่างจากแหล่งน้ำ, ระยะห่างจากถนน และความลาดชันของพื้นที่ จุดข้อมูลการปรากฏของช้างเอเชียถูกรวบรวมจากการสำรวจภาคสนามและการติดตาม การวิเคราะห์ถิ่นที่อยู่อาศัยที่เหมาะสมดำเนินการโดยใช้อัลกอริทึม Machine Learning แบบ Random Forest โดยนำจุดปรากฏของช้างมาเปรียบเทียบกับจุดที่ไม่มีการปรากฏ (pseudo-absence points) และตัวแปรเชิงพื้นที่ที่สกัดได้

ผลการศึกษาที่สำคัญ (Key Findings):
แบบจำลอง Random Forest แสดงประสิทธิภาพสูงในการทำนายถิ่นที่อยู่อาศัยของช้างเอเชีย (ค่า AUC อยู่ที่ประมาณ 0.91) ปัจจัยสำคัญที่ส่งผลต่อความเหมาะสมของถิ่นที่อยู่อาศัย ได้แก่ ความหนาแน่นของพืชพรรณ (โดยเฉพาะ NDVI สูง), ระยะห่างจากแหล่งน้ำที่เหมาะสม และประเภทป่าผลัดใบผสม โดยพื้นที่ป่าที่มีพืชพรรณหนาแน่นใกล้แหล่งน้ำและอยู่ห่างจากการรบกวนของมนุษย์เล็กน้อยมีความเหมาะสมสูงสุด แผนที่ถิ่นที่อยู่อาศัยที่สร้างขึ้นแสดงให้เห็นถึงพื้นที่แกนกลาง (Core Habitat) ที่มีความสำคัญและเส้นทางเชื่อมโยง (Corridors) ที่อาจถูกละเลยไปก่อนหน้านี้

จุดเด่นและการประยุกต์ใช้ (Application):
งานวิจัยนี้แสดงให้เห็นถึงศักยภาพของเทคโนโลยีภาพถ่ายดาวเทียม Sentinel-2 ร่วมกับ Machine Learning ในการสร้างแบบจำลองถิ่นที่อยู่อาศัยสำหรับสัตว์ป่าขนาดใหญ่ได้อย่างมีประสิทธิภาพและคุ้มค่า ข้อมูลและแผนที่ที่ได้สามารถนำไปใช้ในการ:
1.  วางแผนการจัดการพื้นที่อนุรักษ์ช้างเอเชียได้อย่างแม่นยำยิ่งขึ้น
2.  ระบุและปกป้องพื้นที่ถิ่นที่อยู่อาศัยที่สำคัญและเส้นทางอพยพ
3.  ประเมินผลกระทบจากการเปลี่ยนแปลงการใช้ประโยชน์ที่ดินต่อถิ่นที่อยู่อาศัยของช้าง
4.  เป็นแนวทางในการติดตามประชากรช้างและลดความขัดแย้งระหว่างคนกับช้างในอนาคต
