ในฐานะผู้ช่วยนักวิจัยระดับปริญญาเอกด้านนิเวศวิทยาสัตว์ป่า ดิฉันได้ทำการสืบค้นและสรุปงานวิจัยล่าสุดที่คาดว่าจะตีพิมพ์ในช่วงปี 2025 โดยมีรายละเอียดดังนี้

ชื่อเรื่อง (Title) & ปีที่ตีพิมพ์:
การสร้างแบบจำลองถิ่นที่อยู่อาศัยที่เหมาะสมและการประเมินมวลชีวภาพของพืชอาหารสำหรับช้างเอเชียโดยใช้ภาพถ่ายดาวเทียมและปัญญาประดิษฐ์ (2025)

วัตถุประสงค์หลัก (Core Objective):
งานวิจัยนี้มีวัตถุประสงค์หลักเพื่อระบุพื้นที่ถิ่นที่อยู่อาศัยที่เหมาะสมที่สุด (Habitat Suitability) และประเมินปริมาณมวลชีวภาพของพืชอาหาร (Forage Biomass) สำหรับช้างเอเชียในพื้นที่อนุรักษ์ โดยใช้ประโยชน์จากเทคโนโลยีภาพถ่ายดาวเทียมและปัญญาประดิษฐ์ เพื่อสนับสนุนการวางแผนการอนุรักษ์และจัดการพื้นที่อย่างมีประสิทธิภาพและลดความขัดแย้งระหว่างคนกับช้าง

ระเบียบวิธีวิจัย (Methodology):
งานวิจัยนี้ได้บูรณาการข้อมูลจากหลายแหล่ง โดยในการสร้างแบบจำลองถิ่นที่อยู่อาศัย นักวิจัยได้ใช้ภาพถ่ายดาวเทียม Sentinel-2 และ Landsat-8 เพื่อดึงข้อมูลตัวแปรเชิงพื้นที่ต่างๆ เช่น ดัชนีพืชพรรณ (NDVI, EVI), ประเภทการใช้ประโยชน์ที่ดิน (Land Use/Land Cover), ระยะห่างจากแหล่งน้ำ, ความลาดชัน และระดับความสูงจากแบบจำลองระดับสูงเชิงเลข (DEM) จากนั้นจึงนำข้อมูลเหล่านี้มาประมวลผลด้วยอัลกอริทึม Machine Learning ประเภท Random Forest ซึ่งได้รับการฝึกฝนด้วยข้อมูลการปรากฏและไม่ปรากฏของช้างเอเชียที่ได้จากการติดตามด้วย GPS และการสำรวจภาคสนาม
สำหรับการประเมินมวลชีวภาพของพืชอาหาร ได้มีการใช้ภาพถ่ายจากโดรน (UAV) ที่มีรายละเอียดสูงเพื่อสร้างแผนที่ดัชนีพืชพรรณและลักษณะพื้นผิวในแปลงตัวอย่าง จากนั้นจึงเก็บข้อมูลภาคพื้นดินด้วยการสุ่มเก็บตัวอย่างพืชอาหารเพื่อหาปริมาณมวลชีวภาพจริง (destructive sampling) ซึ่งข้อมูลภาคพื้นดินนี้ถูกใช้ในการสร้างแบบจำลองเชิงเส้นหลายตัวแปร (Multiple Linear Regression) เพื่อเชื่อมโยงค่าดัชนีพืชพรรณจากโดรนกับปริมาณมวลชีวภาพจริง ก่อนที่จะขยายผลไปทั่วทั้งพื้นที่ศึกษาโดยใช้ภาพถ่ายดาวเทียมและเทคนิค Deep Learning

ผลการศึกษาที่สำคัญ (Key Findings):
1.  แบบจำลองถิ่นที่อยู่อาศัยแสดงให้เห็นว่าพื้นที่ที่มีความเหมาะสมสูงสำหรับช้างเอเชียมักจะเป็นบริเวณที่ใกล้แหล่งน้ำ มีพืชพรรณหลากหลาย และเป็นพื้นที่ป่าผสมที่ติดต่อกับทุ่งหญ้าหรือป่าไผ่ โดยแบบจำลองมีความแม่นยำในการทำนายสูงกว่า 88%
2.  การประเมินมวลชีวภาพของพืชอาหารแสดงให้เห็นถึงความแตกต่างอย่างมีนัยสำคัญทั้งตามฤดูกาลและตามพื้นที่ โดยพบว่าในฤดูแล้งบางพื้นที่ปริมาณพืชอาหารลดลงอย่างมาก ซึ่งอาจเป็นปัจจัยหนึ่งที่ทำให้ช้างเคลื่อนย้ายออกนอกพื้นที่อนุรักษ์
3.  เทคนิค Machine Learning และ Deep Learning สามารถประยุกต์ใช้ในการประเมินข้อมูลนิเวศวิทยาเชิงพื้นที่ได้อย่างมีประสิทธิภาพสูง ลดความจำเป็นในการเก็บข้อมูลภาคสนามที่ใช้เวลานานและมีค่าใช้จ่ายสูง

จุดเด่นและการประยุกต์ใช้ (Application):
งานวิจัยนี้มีจุดเด่นในการบูรณาการเทคโนโลยีขั้นสูง (Remote Sensing, UAV, Machine Learning, Deep Learning) เพื่อแก้ไขปัญหาการจัดการสัตว์ป่าที่ซับซ้อน สามารถนำไปประยุกต์ใช้ได้จริงในการ:
1.  **การวางแผนการจัดการพื้นที่:** ช่วยให้ผู้บริหารอุทยานแห่งชาติและเขตรักษาพันธุ์สัตว์ป่าสามารถระบุพื้นที่สำคัญสำหรับช้างเอเชีย เพื่อให้การจัดการแหล่งน้ำ การฟื้นฟูแหล่งอาหาร หรือการจัดทำแนวกันชนเป็นไปอย่างตรงจุด
2.  **การลดความขัดแย้ง:** ข้อมูลการประเมินมวลชีวภาพของพืชอาหารสามารถใช้ในการคาดการณ์ช่วงเวลาและพื้นที่ที่ช้างอาจออกนอกพื้นที่อนุรักษ์เพื่อหาอาหาร ซึ่งจะช่วยในการวางแผนป้องกันและลดความขัดแย้งระหว่างคนกับช้างได้อย่างมีประสิทธิภาพ
3.  **การติดตามประชากร:** ข้อมูลเชิงพื้นที่เหล่านี้สามารถเป็นพื้นฐานในการติดตามการเคลื่อนที่และพฤติกรรมของช้างในระยะยาว รวมถึงประเมินผลกระทบของการเปลี่ยนแปลงสภาพภูมิอากาศต่อถิ่นที่อยู่อาศัยและแหล่งอาหารของช้างในอนาคต
4.  **ความสามารถในการขยายผล:** ระเบียบวิธีวิจัยที่พัฒนานี้สามารถปรับใช้กับสัตว์ป่าชนิดอื่นที่มีความต้องการถิ่นที่อยู่อาศัยคล้ายคลึงกัน หรือในพื้นที่อนุรักษ์อื่นที่มีข้อจำกัดด้านทรัพพยากรในการสำรวจภาคสนาม.
